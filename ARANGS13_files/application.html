GTPB Application process:

You will need to send us 3 documents in MS Word or PDF formats.

1) Prepare a short CV. Save it using your name as part of the file name (e.g. Surname_Name_ARANGS_CV)

2) Prepare a simple motivation statement (I wish to attend this course because...). Save it using your name as part of the file name (e.g. Surname_Name_ARANGS_MS)

3) Fill an Application form using the template available in either MS Word or RTF formats, and save using your name as part of the file name (e.g. Surname_Name_ARANGS_AF)

4) Send an e-mail to bicourses "at" igc.gulbenkian.pt with the 3 documents in attachment

Application e-mails that DO NOT carry the THREE attachments will NOT be considered

EARLY Acceptance Request

If you come from a remote location and, for this or a similar reason, you need your acceptance decision taken before the specified NORMAL acceptance notification date, please apply NOW, indicate this need at the end of the form and include the word URGENT in the subject line of the e-mail.


A note on accommodation in the vicinity of IGC

Please note that we can provide help in lowering subsistence costs, by trying to book suitable but cheap accomodation in the vicinity of the IGC, on request. Several hotels and an academic residence have agreements with us. Minimum subsistence costs (food and lodging) can be as low as Eur 30.00 per day by using the local residence an purchasing take-away food.

Couse duration: 4 days      Course Fee: Euro 320.00

The fee includes complete documentation in a CDROM, coffee-breaks, etc. For non-members of IGC/ITQB/IBET a simple lunch in our cafeteria is also included.


Payment method:
You can ONLY pay locally by bringing a cheque in the name of FUNDACAO CALOUSTE GULBENKIAN or cash on the first day of the course.
